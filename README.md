# Running the Django application locally

1.  Run `git clone https://github.com/demaiodev/django_blog.git` using your terminal to clone the repo.
2.  Ensure we’re using python 3 (`python –version`).
3.  Create a virtual environment (`python -m venv venv`) then source/activate (on Windows, run `.\venv\Scripts\Activate.ps1`).
4.  Install requirements (`pip install -r .\requirements.txt`).
5.  Create a .env file in the root project directory and paste in the secrets I will provide you.
6.  Run the Django application locally (`python manage.py runserver`).
